## Project Overview

The goal of this project is to practice and integrate core data skills by:
- Cleaning and preparing exchange rate data  
- Analyzing returns, volatility, and temporal patterns (e.g. day-of-week effects)  
- Writing SQL queries for aggregation and analysis  
- Presenting results in an interactive dashboard  

This project is intended as **hands-on training** and portfolio practice.
